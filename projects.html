Amale (Amaz)

This group created a self navigating robot that could maneuver any obstacle course placed in front of it. This project was unique because it was programmed to remember what paths it had taken, so that it would not attempt to travel that path again if it led to a dead end. This way, the robot could quickly navigate large scale mazes because it had the ability to memorize paths.
